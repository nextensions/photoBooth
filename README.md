## Environment
- Support browser ```firefox``` only
- Support smart card (Thai national ID card)
- Support webcam
- use tracking.js
- use WebCard
- use canvas
- React

## Installation
- Client
```
yarn
```
- Server
```
cd server
yarn
```

## Run
- Client Link ```localhost:3000```
```
npm start
```

- Server Link ```localhost:3030```
```
cd server
npm start
```
or
```
node server/index.js
```
